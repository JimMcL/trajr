Release procedure
-----------------

Run devtools::build_win() and wait for the result

Update version number in DESCRIPTION file. See https://semver.org/

Update NEWS.md

Update session info in README.md: devtools::session_info()

Commit and push all changes

Build source package

Run (possibly from RStudio terminal window): cd ..; R CMD check --as-cran trajr_*.tar.gz

Check that travis-ci build completed successfully with no issues (https://travis-ci.org/JimMcL/trajr)

Upload to CRAN (https://cran.r-project.org/). Use the ‘Optional comment’ field on the webform (and not a separate email) to explain how the feedback on previous submission(s) has been addressed.

Once accepted by CRAN, create a new release on GitHub (https://github.com/JimMcL/trajr)

Update version number to x.y.z.9000
